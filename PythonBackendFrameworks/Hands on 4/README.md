# Hands-On 4

## Python Backend Frameworks

### Topic

Flask REST API Development

---

## Objective

To build RESTful APIs using Flask with Blueprints, request validation, CRUD operations, and custom error handling.

---

## Software Used

- Python 3.x
- Flask
- Flask-SQLAlchemy
- Flask-Migrate
- Visual Studio Code
- Postman

---

# Task 1 - Flask Project Setup

## Features Implemented

- Created Flask project structure
- Configured application settings
- Created Blueprint
- Implemented GET endpoint
- Implemented POST endpoint

### API Endpoints

| Method | Endpoint |
|---------|----------|
| GET | /api/courses/ |
| POST | /api/courses/ |

---

# Task 2 - CRUD Operations & Validation

## Features Implemented

- GET all courses
- POST course
- GET course by ID
- PUT course
- DELETE course
- Request validation
- Custom error handling (400, 404, 500)

### API Endpoints

| Method | Endpoint |
|---------|----------|
| GET | /api/courses/ |
| POST | /api/courses/ |
| GET | /api/courses/<id> |
| PUT | /api/courses/<id> |
| DELETE | /api/courses/<id> |

## Result

Successfully developed a Flask REST API using Blueprints with CRUD operations, request validation, and error handling. All API endpoints were tested successfully using Postman.
